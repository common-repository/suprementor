A set of database powered widgets for creating blogs and shops with the Elementor Page and Theme builder.

== Description ==

Supreme Products and Posts for Elementor is a suite of widgets for the Elementor page builder. It includes widgets for displaying blog posts and WooCommerce products on landing pages along with additional widgets for use with the Elementor Theme Builder to create singles and archives.

### Author Info Widget
Displays a card style box using your blogs author information.

### Post Box Widget
Displays a single blog post card for use on landing pages and sidebars.

### Post Category Box Widget
Displays information about a single post category for use on landing pages and sidebars.

### Post Category List Widget
Displays a list of post categories for use in landing pages and archive sidebars.

### Post List Widget
Loops the Post Box widget to create lists and grids of posts for use on landing pages and sidebars.

### Post Slideshow Widget
Creates a snazzy slideshow for posts using and configurable database query.

### Product Category Box Widget
Displays information about a single product category for use on landing pages and sidebars.

Here's a link to [the demo](https://supremepap.com "Supreme PAP Demo") to see the available widgets and skins.

== Installation ==

1. Install and activate Elementor Page Builder.
1. Install and activate activate this plugin.
1. Use the included widgets in pages or with the Elementor Theme Builder.
